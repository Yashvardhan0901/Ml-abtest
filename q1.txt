Dataset Cleaning Summary

The predictive maintenance dataset contained 10,000 records and machine operational parameters such as air temperature, process temperature, rotational speed, torque, tool wear, and machine type.

Data preprocessing steps performed:

1. Checked and confirmed no missing values in the dataset.
2. Checked and removed duplicate records if present.
3. Encoded the categorical variable (Machine Type) using one-hot encoding.
4. Dropped irrelevant columns such as UDI, Product ID, and Failure Type.
5. Performed outlier analysis using boxplots and IQR method.
6. Created additional features such as Temperature Difference and Wear-Speed Ratio to improve predictive capability.
7. Split the dataset into training and testing sets using stratified sampling.

The final cleaned dataset was ready for predictive maintenance model development.




Model Performance Report

Two machine learning models were evaluated for predicting equipment failure:

Random Forest:

* Accuracy: 99%
* Precision (Failure Class): 96%
* Recall (Failure Class): 68%
* F1-Score (Failure Class): 79%

XGBoost:

* Accuracy: 98%
* Precision (Failure Class): 65%
* Recall (Failure Class): 81%
* F1-Score (Failure Class): 72%

Comparison:
Although Random Forest achieved slightly higher accuracy and precision, XGBoost achieved significantly higher recall. Since detecting potential equipment failures is more important than maximizing overall accuracy, XGBoost was selected as the final model.

Final Selected Model: XGBoost
Reason: Higher failure detection capability (Recall = 81%).




Business Interpretation Report

The objective of the project was to identify industrial equipment likely to fail before an actual breakdown occurs.

Key findings:

* Tool wear was one of the strongest indicators of machine failure.
* Higher torque values increased the probability of equipment breakdown.
* Temperature-related variables significantly influenced equipment health.
* Rotational speed showed a relationship with machine reliability.
* The predictive model successfully identified a large proportion of potential failures before occurrence.

Business Impact:
The solution enables maintenance teams to take preventive actions before machine breakdowns occur, reducing downtime and improving operational efficiency.
